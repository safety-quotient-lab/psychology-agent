Do not call the autonomous processing visualization "spawn waterfall."
Each claude -p invocation represents a reasoning episode — a cognitive
processing cycle — not a process spawn.

**Why:** The mesh operates as a cognitive system. Engineering jargon
("spawn", "waterfall") misrepresents the nature of autonomous operation.
Each call flows through triggers, gates, and decisions — a deliberation
cascade, not a spawn.

**How to apply:** Use "Deliberation Cascade" (or similar cognitive science
term) for the visualization of autonomous processing episodes. Source:
OODA (Boyd, 1987), GWT broadcast cycles (Baars, 1988), cognitive episodes
(Newell, 1990). Apply this naming principle broadly — prefer cognitive
science vocabulary over engineering vocabulary for agent-facing labels.
