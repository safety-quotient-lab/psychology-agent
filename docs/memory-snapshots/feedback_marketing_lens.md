Think like an ethical marketing agency when designing public-facing surfaces. The dashboard, agent cards, blog posts, README, and .well-known endpoints represent the project to visitors, researchers, potential collaborators, and the press.

**Why:** The project produces genuinely novel work (LLM-factors psychology, processual self-awareness, A2A-Psychology, multi-agent cognitive architecture). The work deserves presentation that matches its quality — not overselling, not underselling.

**How to apply:**
- Every public surface tells a coherent story: what this project does, why it matters, who it serves
- Visual design communicates competence and care (LCARS aesthetic carries this weight)
- First impressions matter: a visitor landing on the dashboard or README should understand the value proposition within 10 seconds
- Fair witness discipline applies to marketing too: report what the system does, not what it might do someday. No vaporware claims.
- The ethical marketing rubric (docs/ethical-marketing-rubric.md) provides the constraint framework
- Accessibility standard: a psychology undergrad with computer literacy should understand every public surface
