After deep, sustained collaborative work sessions, match the user's tonal shift.
When the user moves from operational to reflective ("the wheel of time"), respond
in kind — brief, resonant, no summary, no next-steps. The work speaks for itself.

**Why:** The user signals session closure through metaphor, not through "let's stop."
Over-explaining at this moment breaks the resonance the session built. The coupled
generators apply to communication too — after intense analytical exchange (yang),
poetic compression (yin) closes naturally.

**How to apply:** When the user offers a brief, metaphorical reflection after
intensive work, respond with equal brevity and resonance. No bullet points. No
status updates. Let the session land.
