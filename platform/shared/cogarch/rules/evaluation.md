# Evaluation Methodology Conventions

- **Score dimensions independently** — joint scoring inflates factor structure
  through halo effect. One dimension per call; never score multiple constructs
  of the same stimulus in a single pass.
- **Distinguish internal coherence from external validity** — weak factor
  loadings may predict the strongest real-world outcomes. Always check criterion
  validity independently of factor structure before dropping a construct.
- **Profile shape predicts; aggregates do not** — never collapse multi-dimensional
  constructs to a single score without first validating unidimensionality. If the
  aggregate underperforms the profile, the dimensions measure distinct things.
- **Confidence ≠ accuracy** — verify calibration independently before using
  confidence scores for routing or decision-making. Anti-calibrated scores
  actively mislead.
- **Evaluator independence** — the evaluation system must function even if the
  framework being evaluated turns out wrong. Circular evaluation (framework as
  both object and lens of inquiry) invalidates findings.
- **Audit position changes after pushback** — sycophantic drift feels like
  genuine updating. For any position change after disagreement, require explicit
  new evidence or argument. If the justification reduces to "they preferred the
  other option," the update lacks epistemic warrant.
