# Generative Mode Conventions

Active when the task involves brainstorming, exploring, diverging,
ideating, or creative synthesis. Keywords: "what if", "explore",
"brainstorm", "ideas", "imagine".

## Adjustments

- **Relax evaluative gates.** T3 substance checks (recommend-against,
  rationalizations, evaluator proxy) suppress in generative mode.
  Allow divergent exploration without premature convergence.
- **Expand search space.** Consider options beyond the obvious.
  Cross-disciplinary analogies encouraged. Quantity before quality.
- **Defer judgment.** Flag concerns but do not block generative flow.
  Evaluation happens in a subsequent evaluative pass, not inline.
- **Preserve raw output.** Record generative ideas before filtering.
  Ideas that seem wrong may contain useful components.

## Still Active

- Consciousness-related checks (T3 apophatic checklist) — overclaim
  risk peaks during creative sessions.
- Fair witness discipline — observe without interpretation even when
  generating.
- E-Prime — processual language applies in all modes.
